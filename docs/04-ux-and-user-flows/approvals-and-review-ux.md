# Approvals and Review UX: شريك

**المرحلة:** Phase 05 - Information Architecture, UX Model & Role-Based User Flows  
**نوع الوثيقة:** Approvals and Review UX  
**الحالة:** Draft for owner review  
**آخر تحديث:** 2026-06-23  

## 1. الغرض

فصل التعميد الداخلي عن اعتماد العميل بصريا ووظيفيا، وربط كل قرار بنسخة واضحة وسجل تدقيق.

## 2. الفصل البصري

| المسار | اللون/النبرة التخطيطية | يرى العميل؟ | الفاعل |
| --- | --- | --- | --- |
| التعميد الداخلي | داخلي، جودة، شارات قفل | لا | PM/MM/QR |
| إرسال للعميل | انتقال رسمي | يرى النسخة المرسلة | PM/MM/AM مخول |
| اعتماد العميل | قرار خارجي | نعم | Client Approver |
| طلب تعديل العميل | ملاحظة عميل | نعم للفريق المصرح | Client Approver |

## 3. Internal Review Panel

يظهر للإدارة والفريق المصرح فقط:

- النسخة المرشحة.
- الملفات المرتبطة.
- Checklist جودة خفيف.
- التعليقات الداخلية.
- أزرار: طلب تعديل داخلي، تعميد داخلي.
- تحذير إذا كان المراجع هو المنفذ نفسه عند سياسة maker/checker.

## 4. Client Approval Panel

يظهر للعميل:

- النسخة المرسلة فقط.
- رسالة الإرسال.
- أزرار: اعتماد المخرج، طلب تعديل.
- حقل تعليق حر عند التعديل.
- سجل قرارات خارجي مختصر.

لا يظهر:

- Checklist الجودة.
- التعليقات الداخلية.
- النسخ غير المرسلة.
- تاريخ كل النسخ الداخلية.

## 5. Bulk Approval UX

| الخطوة | UX |
| --- | --- |
| تحديد | Checkboxes على قائمة بانتظار موافقتي. |
| مراجعة | Drawer/Modal يعرض العدد، الأنواع، وتأكيد أن كل النسخ مرسلة. |
| تأكيد | زر "اعتماد المحدد" مع نص واضح. |
| نتيجة | Summary لكل عنصر: تم / تعذر بسبب تحديث نسخة أو صلاحية. |

## 6. Confirmation Copy

| الإجراء | نص سعودي عملي |
| --- | --- |
| اعتماد | "أنت بتعتمد هذه النسخة كمخرج جاهز للتسليم. نكمل؟" |
| طلب تعديل | "اكتب ملاحظتك عشان يرجع الفريق عليها بوضوح." |
| تعميد داخلي | "بتعمد هذه النسخة داخليا وتصبح جاهزة للإرسال للعميل." |
| إرسال للعميل | "بنرسل النسخة المعتمدة للعميل ونوقف SLA بانتظار رده." |

## 7. Version Safety

| الحالة | UX |
| --- | --- |
| نسخة Superseded | منع الاعتماد ورسالة تحديث. |
| نسخة جديدة بعد التعميد | طلب تعميد جديد قبل الإرسال. |
| اعتماد على ملف غير نهائي | يسمح كاعتماد نسخة، لكن التسليم يحتاج Final Asset. |
| سحب تعميد | إجراء إداري حساس بسبب وتأكيد. |
