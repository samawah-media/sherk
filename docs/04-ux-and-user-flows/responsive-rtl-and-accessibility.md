# Responsive, RTL, and Accessibility: شريك

**المرحلة:** Phase 05 - Information Architecture, UX Model & Role-Based User Flows  
**نوع الوثيقة:** Responsive, RTL, and Accessibility Planning  
**الحالة:** Draft for owner review  
**آخر تحديث:** 2026-06-23  

## 1. الغرض

تحديد متطلبات التخطيط لـ RTL، الجوال، وWCAG 2.2 AA كهدف تصميمي.

## 2. RTL-First

| المجال | القاعدة |
| --- | --- |
| اتجاه الواجهة | `dir=rtl` أساسا. |
| النصوص | عربية سعودية ودودة وعملية. |
| التواريخ | عرض عربي/سعودي مفهوم، مع حفظ المفاهيم الزمنية لاحقا. |
| الأرقام | دعم الأرقام العربية أو اللاتينية حسب إعداد، دون كسر الجداول. |
| الأيقونات | اتجاه الأسهم والتسلسل يناسب RTL. |

## 3. Responsive Priorities

| الدور | الأولوية |
| --- | --- |
| العميل | Mobile-first مع دعم Desktop. |
| الفريق | Desktop-first لكن كل شيء مهم يعمل من الجوال. |
| الإدارة | Desktop-first، والجوال للقراءة والتدخلات العاجلة. |
| Platform Admin | Desktop-first. |

## 4. Mobile Patterns

| الشاشة | الجوال |
| --- | --- |
| بانتظار موافقتي | قائمة بطاقات مع CTA ثابت. |
| تفاصيل المخرج | صفحة كاملة بدل Drawer إذا ضاق العرض. |
| Kanban | قائمة مراحل لا سحب أفقي إلزامي. |
| ملفات | Grid بسيط أو List حسب النوع. |
| Activity Feed | فلاتر collapse. |

## 5. WCAG 2.2 AA Planning Checklist

| المعيار | تطبيق UX |
| --- | --- |
| Keyboard Navigation | كل إجراء له بديل بدون Drag. |
| Focus Management | Drawer/Modal يحبس التركيز ويعيده. |
| Contrast | لا تعتمد على ألوان باهتة فقط. |
| Status not by color | نص وأيقونة وشارة. |
| Error Association | رسائل الخطأ قرب الحقول. |
| Touch Targets | أزرار الجوال لا تقل عن مساحة مريحة. |
| Reduced Motion | الحركات قابلة للتقليل لاحقا. |
| Arabic Typography | أحجام مناسبة لا تعتمد على viewport فقط. |
| Date/Number Localization | عرض مفهوم للمستخدم السعودي. |

## 6. Accessible Drag and Drop

- كل كارت له قائمة "نقل إلى".
- الانتقالات غير المسموحة تظهر سببها.
- قارئ الشاشة يسمع الحالة الحالية والهدف.
- لا ينجح السحب إذا كسر التعميد أو التسليم.

## 7. Permission Denied UX

| الحالة | الرسالة |
| --- | --- |
| عميل خارج النطاق | "ما عندك صلاحية للوصول لهذا المحتوى." |
| Viewer يحاول اعتماد | "حسابك مخصص للمشاهدة فقط." |
| فريق غير مسند | "هذا المخرج خارج نطاقك الحالي." |
| دعم منصة بلا جلسة | "يتطلب الوصول جلسة دعم موثقة." |
